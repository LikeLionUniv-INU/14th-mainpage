## ✨ Key Features

1.  **Home:**
    - 모집 기간 계산, 메뉴 항목 제시
2.  **Tracks:**
    - 4개 파트(기획/디자인/프론트/백엔드) 소개
3.  **Curriculum:**
    - 연간 일정 타임라인
4.  **FAQ:**
    - 자주 나오는 질의응답
5.  **Apply:**
    - 모집 기간에 따른 지원 받기, 신청 폼으로 이동
